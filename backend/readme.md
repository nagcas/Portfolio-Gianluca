# Backend del Portfolio

Questo Ã¨ il backend del portfolio di Gianluca Chiaravalloti, sviluppato con **Node.js** e **Express.js**. Il backend gestisce le richieste del frontend e le connessioni al database MongoDB.

## Tecnologie utilizzate

- **Node.js**: Ambiente di esecuzione JavaScript lato server.
- **Express.js**: Framework per la creazione di API e server web.
- **MongoDB**: Database NoSQL per la gestione dei dati.
- **Mongoose**: ODM per MongoDB, utilizzato per la modellazione dei dati.
- **Nodemailer**: Libreria per l'invio di email.
- **dotenv**: Gestione delle variabili d'ambiente.
- **CORS**: Middleware per la gestione delle richieste cross-origin.
