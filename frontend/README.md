# Frontend del Portfolio

Questo è il frontend del portfolio di Gianluca Chiaravalloti, sviluppato con **React.js** e **Vite**. Il design è responsive e si basa su **Bootstrap** per uno sviluppo rapido e flessibile.

## Tecnologie utilizzate

- **React.js**: Libreria per la creazione di interfacce utente dinamiche.
- **Vite**: Tool per lo sviluppo front-end con ottime performance.
- **Bootstrap 5**: Framework CSS per layout responsive.
- **AOS**: Libreria per le animazioni on-scroll.
- **React Router**: Gestione delle rotte nel frontend.
- **i18next** e **React-i18next**: Librerie per la gestione della localizzazione (i18n).
- **ESLint**: Tool per il controllo della qualità del codice.
